# MindFlow Journal Platform

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

MindFlow Journal Platform is a full-stack journaling application designed to help users capture their thoughts, experiences, and emotions effortlessly. It combines the simplicity of a modern journal app with advanced AI-powered sentiment analysis, giving you insights into your emotional patterns over time.  

Whether you want to track personal growth, reflect on daily events, or organize your thoughts, MindFlow provides a visually engaging, responsive, and interactive environment to make journaling both meaningful and enjoyable.

---

## Features

- Add, edit, and delete journal entries with ease  
- Automatic sentiment analysis: Positive, Negative, Neutral  
- Animated sentiment chart to visualize your emotional trends  
- Organized list of past entries for easy review  
- Calm, pastel-themed UI with responsive design for desktop and mobile  
- Dashboard to quickly view all journal entries
